O que é o plugin?

Basicamente se trata de um slideshow de imagens automático que além de ser leve (5.29 KB), pode ser facilmente configurado.

Como faz para utilizá-lo?

Primeiramente Acesse esse link para baixar o plugin:

Depois de baixado, copie os arquivos das pastas CSS e JS para o seu projeto.

Demonstração do código para utilizá-lo

Copie do arquivo "INDEX.HTML" os seguintes códigos:
Primeiro dentro de "Head" do seu código, insira onde colocou o arquivo CSS que foi baixado. Não esqueça de referenciar o arquivo jquery padrão.

<link rel="stylesheet" type="text/css" href="css/index.css">
<script src="js/jquery-3.3.1.js"></script>

Em seguida insira esse código ainda dentro de "HEAD"

<style>
    body {
      background-color: #ffffff;
      text-align: center;
      color: #000000;
      font-family: 'Roboto';
      min-height: 100vh;
    }
    * {
      margin: 0;
      padding: 0;
    }
    img {
      vertical-align: top;
    }
    .clearfix::after {
      content: "";
      display: table;
      clear: both;
    }
    ul li {
      list-style: none;
    }
    .banner {
      display: inline-block;
    }
    #banner1 h1 {
      position: absolute;
      width: 100%;
      height: 50px;
      bottom: 0;
      left: 0;
      background-color: rgba(0, 0, 0, 0.5);
      line-height: 50px;
      color: yellow;
    }
    #ctrl {
      text-align: center;
    }

  </style>

Agora Insira essa parte do código de "INDEX.HTML" em "BODY" do seu arquivo html. Essa parte se trata das imagens do slideshow.

Por fim Insira esse código em "BODY" do seu código. Esse é o menu de controle do slide, não esqueça de colocar o endereço do arquivo myscroll.js

<script src="js/myscroll.js"></script>
<script>
$(function() {
  $('#banner1').myscroll({
      picEl: $('#move'),
      ctrlEl: $('#ctrl'),                                         // opcoes de controle
      libs: true, //cria aquele pontinho em baixo, na transicao   | true / false
      arrows: true, //  flechas para passar de slide              | true / false
      autoPlay: true,  //passar de slide automaticamente          | true / false
      time: 3000,      //tempo que fica parado na tela            | 0 ate o que vc quiser
      speed: 1000 ,     //velocidade da transicao                 | 0 ate o que vc quiser
      effect: 'left', //  efeitos de transicao                    | top left fade
      hover: true  // flechas aparecem com o mouse na imagem      |true / false
  });
})
</script>
